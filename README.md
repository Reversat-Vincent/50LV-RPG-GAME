# 50LV RPG GAME
Second projet de jeu en Python

>Il s'agit d'un petit jeu vidéo sur un thème de fantasy.
>
>Le but de ce jeu est d'atteindre le cinquantième stage et d'y vaincre le dragon.
>
>Le joueur dispose de huit personnages, il est libre d'en utiliser plusieurs ou non mais les combats s'effectuent uniquement avec un seul personnage séléctionné. Un personnage dont les points de vie atteingnent zéro ne sera plus utilisable.
>
>Pour regagner des points de vie, le joueur possède quelques potions à efficacité variable (10PV, 20PV, 30PV, 60PV) en quantité limitée (20, 15, 10, 5). L'inventaire des potions est commun à tous les personnages.
>
>Pour atteindre le stage cinquante, le joueur doit d'abord conquérir tous les étages un par un. Conquérir un étage débloque le suivant mais n'empèche pas le joueur de continuer de gagner des niveaux dans les étages inférieurs. La liste des stages est à défilement vertical.
>
>Gagner un niveau augmente les PV maximum du personnage et lui offre entre 2 et 3 points de statistique que le joueur est libre de répartir à sa guise.
>
>Les ennemis possèdent tous une répartition des statistiques différentes. Des boss se trouvent dans les étages 10, 20, 30, 40 et 50.


J'ai développé ce jeu avec deux objectifs en tête :
- Découvrir les classes en Python
- M’entraîner à utiliser Pygame

Les images utilisées dans ce projet proviennent de trois sources différentes :
- [Fire Emblem Heroes](https://www.spriters-resource.com/mobile/fireemblemheroes/) (fonds, chiffres et phases)
- [The Battle for Wesnoth](http://wesnoth.org/) (boutons d'interface)
- [Ressources graphique de RPG Maker](https://www.rpgmakerweb.com/additional-downloads) (personnage)

Développé sous Linux en Python 2.7.12, nécessite Pygame
